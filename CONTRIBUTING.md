# 贡献指南

感谢你愿意为 NEFU Course Resources 贡献资料。

## 可以贡献的内容

- 自己整理的课程笔记；
- 历年试卷或回忆版试题；
- 复习资料、知识点总结、题型整理；
- 实验报告模板、实验说明、参考报告；
- 课程设计报告、项目代码、说明文档；
- 选课建议、备考经验、学习路线；
- 开源教材、公开课程、参考网站索引。

## 不建议上传的内容

- 未授权的教材扫描版、盗版电子书；
- 付费课程、付费资料包或网课资源；
- 明确标注禁止外传的课件、试卷或资料；
- 包含姓名、学号、手机号、身份证号等个人隐私的信息；
- 与课程学习无关的内容。

## 目录放置规则

优先按照以下路径放置：

```text
resources/学院/专业/课程/资料类型目录/
```

公共基础课可以放在：

```text
resources/公共基础课程/课程/
```

如果不确定资料应该放在哪里，可以先发 Issue 询问。

## 文件命名规范

推荐格式：

```text
年份-课程名-资料类型-补充说明-贡献者.扩展名
```

示例：

```text
2023-数字电子技术-期末试卷-A卷-匿名.pdf
2024-模拟电子技术-期末复习笔记-匿名.pdf
2022-C语言程序设计-实验报告模板-佚名.docx
2024-单片机原理-课程设计-智能小车-by-昵称.md
```

如果不想暴露真实姓名，可以使用 `匿名`、`佚名` 或 `by-昵称`。

## Pull Request 说明

提交 Pull Request 时，请说明：

- 资料所属学院或课程类别；
- 资料所属专业；
- 资料所属课程；
- 资料类型；
- 资料来源；
- 是否已经脱敏；
- 是否确认可以公开分享。

维护者会重点检查路径、命名、隐私、版权风险和重复内容。
