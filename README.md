# NEFU Course Resources

东北林业大学课程资料开源归档仓库。

本项目面向东北林业大学学生，集中整理可公开分享的课程笔记、历年试卷、复习资料、实验报告、课程设计、学习经验和公开资源索引。项目目标是让同学能够按统一路径快速找到资料，也让后续贡献者知道资料应该放在哪里、如何命名、哪些内容不适合公开上传。

## 快速查找

资料按照“课程入口 + 固定资料类型”的方式组织：

```text
resources/课程类别或学院/专业/课程/
```

首批收录范围：

- 公共基础课程
- 控制与信息工程学院 / 电子信息工程
- 通识选修课程

## 资料类型

每门课程建议使用以下固定目录：

| 目录 | 内容 |
|---|---|
| `01-课程笔记` | 课堂笔记、手写笔记、知识点整理 |
| `02-课件讲义` | 授权课件、公开讲义、课程资料 |
| `03-历年试卷` | 期末试卷、补考试卷、回忆版试题 |
| `04-复习资料` | 重点整理、速通提纲、题型总结 |
| `05-实验报告` | 实验模板、实验说明、参考报告 |
| `06-作业答案` | 作业参考、习题答案、题解 |
| `07-课程设计` | 课程设计报告、项目代码、说明文档 |
| `08-学习经验` | 选课建议、备考经验、避坑指南 |
| `09-开源项目` | 课程相关代码、仿真工程、嵌入式工程 |
| `99-资源索引` | 推荐教材、公开课、参考网站 |

## 如何贡献

欢迎通过 Pull Request 贡献资料。提交前请先阅读：

- [贡献指南](CONTRIBUTING.md)
- [资料命名规范](docs/资料命名规范.md)
- [版权与删除说明](docs/版权与删除说明.md)

贡献资料前请确认：

- 资料路径正确；
- 文件名符合规范；
- 文件已经移除姓名、学号、手机号等个人隐私信息；
- 资料没有明显版权风险，也没有被明确禁止公开传播。

## 版权与免责声明

本项目仅用于学习交流与资料归档。请勿上传未授权教材扫描版、盗版电子书、付费课程资料、包含个人隐私的信息，或其他不适合公开传播的内容。

如你认为仓库中的内容侵犯了你的合法权益，请通过 Issue 联系维护者，我们会尽快核实并处理。
