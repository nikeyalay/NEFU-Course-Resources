# 贡献指南

本页是对仓库根目录 `CONTRIBUTING.md` 的补充说明。

## 推荐流程

```text
Fork 仓库
-> 按目录放入资料
-> 按命名规范重命名
-> 补充课程 README 或 course-meta.yml
-> 提交 Pull Request
-> 维护者审核
-> 合并进入主仓库
```

## 审核重点

维护者会重点检查：

1. 文件路径是否正确；
2. 文件名是否符合规范；
3. 是否包含个人隐私信息；
4. 是否存在明显版权问题；
5. 是否重复上传；
6. 是否和课程学习相关。

## 课程目录缺失怎么办

如果你要贡献的课程还不存在，可以复制 `templates/README.course.md` 和 `templates/course-meta.yml` 创建新课程目录。课程目录命名建议使用课程正式名称。
