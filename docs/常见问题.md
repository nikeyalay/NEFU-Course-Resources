# 常见问题

## 可以上传教材 PDF 吗？

不建议上传未经授权的教材扫描版或电子书。可以在 `99-资源索引` 中记录书名、作者、出版社、ISBN、官网链接或图书馆馆藏链接。

## 可以匿名贡献吗？

可以。文件名中的贡献者字段可以写 `匿名`、`佚名` 或 `by-昵称`。

## 可以上传老师课件吗？

只有在确认课件允许公开分享时才建议上传。如果课件明确禁止外传，或无法确认授权，请不要上传。

## 资料里有姓名和学号怎么办？

请先脱敏，删除或遮盖姓名、学号、手机号、身份证号等个人信息后再提交。

## 课程目录不存在怎么办？

可以参考 `templates/README.course.md` 和 `templates/course-meta.yml` 新建课程目录，也可以先提交 Issue 请求维护者补充。
