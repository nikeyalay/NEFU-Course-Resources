# 版权与删除说明

本项目鼓励分享可公开传播的学习资料，但不鼓励上传版权或公开权限不明确的内容。

## 不直接上传的内容

- 未授权教材扫描版；
- 盗版电子书；
- 付费课程资料；
- 明确禁止外传的课程资料；
- 包含个人隐私的文件；
- 与课程学习无关的文件。

## 教材和电子书如何处理

对于教材、电子书、网课等资料，推荐只放索引：

```text
书名、作者、出版社、ISBN、图书馆馆藏链接、官网链接、公开课程链接
```

索引文件建议放在课程目录下的：

```text
99-资源索引/
```

## 删除请求

如果你认为仓库中的内容侵犯了你的合法权益，请通过 Issue 提交删除请求，并说明：

- 文件路径；
- 删除原因；
- 你与该资料的关系；
- 可用于核实的信息。

维护者会尽快核实并处理。
