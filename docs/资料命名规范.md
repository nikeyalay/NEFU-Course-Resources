# 资料命名规范

资料文件推荐使用以下格式：

```text
年份-课程名-资料类型-补充说明-贡献者.扩展名
```

示例：

```text
2023-数字电子技术-期末试卷-A卷-匿名.pdf
2024-模拟电子技术-期末复习笔记-匿名.pdf
2022-C语言程序设计-实验报告模板-佚名.docx
2024-单片机原理-课程设计-智能小车-by-昵称.md
2023-信号与系统-期末回忆版-匿名.md
```

如果无法确认资料年份，可以使用 `未知年份`：

```text
未知年份-人工智能-课件-第01章绪论-匿名.ppt
```

## 字段说明

| 字段 | 说明 |
|---|---|
| 年份 | 资料对应年份，例如 `2024` |
| 课程名 | 使用课程正式名称或常用名称 |
| 资料类型 | 例如 `期末试卷`、`复习笔记`、`实验报告模板` |
| 补充说明 | 例如 `A卷`、`第1章`、`智能小车` |
| 贡献者 | 可写真实姓名、昵称、`匿名` 或 `佚名` |

## 不推荐命名

```text
最终版.pdf
复习资料.pdf
老师发的.zip
超级重点.docx
新建文件夹/
```

这些名称无法体现年份、课程、资料类型和来源，后续很难检索和维护。
